# Run and deploy your AI Studio app

This repository contains everything you need to run your app locally today, with a lightweight structure for future updates.

View your app in AI Studio: https://ai.studio/apps/fe142a7f-5334-4e8c-b4fb-57d79c817a0f

## Run locally

**Prerequisites:** Node.js

1. Install dependencies: `npm install`
2. Set `GEMINI_API_KEY` in `.env.local` to your Gemini API key
3. Start the app: `npm run dev`
